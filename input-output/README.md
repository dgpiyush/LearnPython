# input and output 

## input 

- for input python have function -  ` input() ` 
- it takes input msg as param - ` input(msg) `


## ouput

- for output python have function - ` print() `
- it takes msg as param to print -  ` print(msg) `
- it can take any object, str, number, variable.

example:
 ```

    # simple string msg 
    print("hello world")

    # number 
    print(4)

    # variable
    num = 5
    print(num)  
```
